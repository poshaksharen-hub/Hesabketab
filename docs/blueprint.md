# **App Name**: Hesab Ketab

## Core Features:

- Data Collection: Collect and categorize financial data for each user.
- Data Sharing: Enable data sharing and joint accounting between two users.
- Database Integration: Store and manage user financial data using Firestore.
- Personalized insights: Provide personalized financial insights and recommendations using generative AI based on transaction history. The tool uses transaction history to determine which insights are most helpful.

## Style Guidelines:

- Primary color: Deep saffron (#FF9933), capturing both tradition and modernity.
- Background color: Pale cream (#FFF3E0), lending a sense of warmth.
- Accent color: Burnt orange (#CC6633), adding contrast to the main interactive elements.
- Headline font: 'Alegreya', serif, offering a mix of contemporary and elegant feel.
- Body font: 'PT Sans', sans-serif, for a modern and approachable look. Pairing with 'Alegreya'.
- Use minimalistic icons with a line style for financial transactions and categories.
- Implement subtle transition animations for data updates and navigation.
